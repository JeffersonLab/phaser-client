Phaser Client

The Phaser Swing Client is an application for Operators at JLab to control the Phaser server.

Usage

The Phaser Swing Client is distributed as an executable jar file. It was developed as a certified product and Linux binaries can be found in the /cs/certified/apps/phaser file system path. The Phaser Swing Client requires Java 7 or greater and can be launched on Linux with the following command: 


./phaser-swing-client.sh
Or to have debugging logged to the console: 

./debug-phaser-swing-client.sh
Or to invoke Java manually (without the bash shell's Java version check): 

java -jar phaser-swing-client.jar

The versioning is organized into "major.minor" where major versions indicate API changes and minor versions are for bug fixes and implementation changes.

Configuration

A Java properties file named client.properties is located in the config directory. Users can set the host and port of the Phaser server here. The URL to documentation is also specified here.

Source Code

The Java Phaser Swing Client application is named phaser-swing-client and is stored in the git repository on devl00. It can be accessed by members of the JLab accelerator group epics from an accelerator Linux machine with the command:


git clone ssh://devl00/usr/devsite/git/phaser-swing-client.git

An Apache Ant build file (build.xml) is used to compile and package the application.
