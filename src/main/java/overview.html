Phaser Client

The Phaser Swing Client is an application for Operators at JLab to control the Phaser server.

Usage

The Phaser Swing Client is distributed as an executable jar file. It was developed as a certified product and Linux binaries can be found in the /cs/certified/apps/phaser file system path. The Phaser Swing Client requires Java 11 or greater and can be launched on Linux with the following command: 


./phaser-client

Configuration

A Java properties file named client.properties is located in the config directory. Users can set the host and port of the Phaser server here. The URL to documentation is also specified here.

Source Code

The Java Phaser Swing Client application is named phaser-client and is stored on GitHub. It can be accessed with the command:


git clone https://github.com/JeffersonLab/phaser-client.git
