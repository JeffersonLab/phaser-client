Release notes for Phaser Swing Client

Version 0.4 - July 30, 2014

This is the initial release of the Phaser Swing Client. This client provides a graphical user interface for operators and scientists to control a cavity phasing job, monitor the job, and review the results.

Version 0.14 - March 18, 2016

Updated to match Requirements 3.1 and Client-Server API 1.17. Main changes include removal of continuous and increment correction and adding the ability to apply corrections at a later date.

Version 0.16 - March 28, 2016

Updated to match Client-Server API 1.18. Main changes include inclusion of server version string in help dialog.

Version 1.0 - April 5, 2016

Initial certified release. Documentation cleaned up.

Version 1.1 - October 25, 2016

min phase error filter now considers the magnitude (absolute value) so that negative error is considered properly.

Version 1.2 - March 22, 2018

max momentum error (dp/p) field now allows values in range 0.000001 - 1, whereas before the range was 0.001 - 9999+

Version 1.3 - March 27, 2018

Export phaser results to elog feature added

Version 1.4 - April 11, 2018

Last Corrected Date added to New Job dialog cavity selection table and cavities are sorted by oldest correction first, by default. Sorting can be toggled by clicking table column header.

Version 1.4.1 - Oct 15, 2018

Fixed classpath to include elog library and bumped elog lib to version 3.1.0
